## 🚀 快速开始

### 方法一：本地运行

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 打开浏览器访问 http://localhost:5173
```

### 方法二：部署到 Vercel（推荐）

1. 将此文件夹上传到 GitHub
2. 访问 https://vercel.com
3. 用 GitHub 登录
4. 点击 "Import Project" 选择你的仓库
5. 点击 "Deploy"
6. 等待几分钟，获得免费网址！

### 方法三：部署到 Netlify

1. 将此文件夹上传到 GitHub
2. 访问 https://netlify.com
3. 用 GitHub 登录
4. 点击 "Add new site" → "Import an existing project"
5. 选择你的仓库
6. Build command: `npm run build`
7. Publish directory: `dist`
8. 点击 "Deploy"
